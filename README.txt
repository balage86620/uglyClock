<! -- SETUP UGLY CLOCK --!>
1. Copy 'uglyClock.scr' to '...\WINDOWS\SYSTEM32
2. Press the Windows key, type Change screen saver, then press Enter.
3. On the Screen Saver Settings window in 'Screen saver' box click the down arrow.
4. Select uglyClock.
5. First click Apply and then OK.

<! -- REMOVE UGLY CLOCK --!>
1. On the Screen Saver Settings window select another screen saver.
2. Remove file 'uglyClock.scr' from '...\WINDOWS\SYSTEM32'
3. Remove file 'uglyClock.scr' from '...\WINDOWS\SYSWOW64'

<! -- ABOUT UGLY CLOCK --!>
Simple screen saver for Windows. It write the local time in any seconds with its ugly characters.
Screen saver stop, when clicked or a key pressed.
